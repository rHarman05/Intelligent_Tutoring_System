# Intelligent Tutoring System for Chemistry

An adaptive Chemistry tutoring applications that personalizes questions based on student performance using an IRT-based model.

## Features
- Student login and progress tracking
- Adaptive questions selection
- Lesson modules with visuals
- Performance analytics
- SQLite database backend

## Installation (Local Build)
pip install -r requirements.txt
python chemistry_its.py

## Run without Python
Download the .exe from the Releases sections and double click to run.

## Tech Stack
Python, Tkinter, SQLite, NumPy, Matplotlib
